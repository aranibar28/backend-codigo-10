### Create and activate virtual enviroment (Git Bash)

    python -m venv env
    . env/Scripts/activate

### Install app dependencies

    pip install -r requirements.txt

### Start Server with Flask
    
    flask run
 
### Start Server with Django
    
    python manage.py runserver
